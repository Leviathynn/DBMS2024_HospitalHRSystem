Information on how to use this website

This site uses publically available medical information and user inserted information to calculate the cost of various medical expenses and insurance covereages based on multiple factors.

The site is still in it's alpha version. This site should also not be an only source of information, and it is recommended to speak to a professional regarding possibly life altering decisions.

This site is split into 3 sections.

Information
- Info page has a summary of the application and it's usage.
Calculations
- Takes in the users information and circumstances, then generates an estimate of the cost of the procredure. Includes the history of past inputs up to the previous five inputs.
References
- Reference page of the sources and data used in this application.

For information on set up and details please go to the program's README.md
